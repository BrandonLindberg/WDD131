Weather Engine Site Plan

Brandon Lindberg

WDD 131-02

Overview

Purpose

My purpose is to create a site that displays current and past trends of weather conditions for any city, and displays it in graphs.

Audience

Anyone interested in current weather conditions and trends being visualized in graph form.

Dynamic elements

JavaScript will be used to pull in the current weather conditions of a searched location and generate graphs based off of those conditions.

Content

Home page

The home page will consist of a simple header and menu for About and Help pages. It will have a field to search for any location, and it will display current extreme conditions from around the world as well. The site will be fairly minimalistic, and aside from the condition graphics and graphs, there will not be many graphical elements

Location Details

The location details page will consist of the current conditions of the search location, as well as a forecast graph and a history graph. The graphs will be made with Chart.js
